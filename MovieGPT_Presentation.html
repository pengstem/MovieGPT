项目介绍：MovieGPT 是一个 AI 驱动的对话界面，用于使用自然语言查询 IMDb 电影数据库。该系统通过 Google Gemini AI 将用户问题转换为 SQL 查询，并针对包含 IMDb 数据集的 MySQL 数据库执行这些问题。该应用程序提供基于 React 的聊天界面，具有实时流式响应和集成的电影信息显示。

🗄️ 数据库设计
基于 IMDb 数据集的 MySQL 架构
🐳 Docker
🐬 MySQL 8.4
🎬 IMDb Dataset
📊 7 Core Tables

🎭 title_basics

电影/电视剧基本信息，包含标题、类型、发行年份、时长、类别等核心数据

⭐ title_ratings

评分数据，包含平均评分和投票数，用于电影质量和热度排序

👥 title_principals

主要演员和制作人员信息，支持按角色查询相关电影

🎬 title_crew

导演和编剧信息，支持按制作人员查询作品

👤 name_basics

人员基本信息，包含姓名、出生年份、职业等个人数据

🌍 title_akas

电影别名和多语言标题，支持跨语言查询

🐳 Docker 部署优势： 一键启动、环境隔离、数据持久化、自动初始化 IMDb 数据集
⚙️ 后端设计
FastAPI + AI 驱动的服务端架构
🐍 Python 3.13+
⚡ FastAPI
🤖 Google Gemini
🎬 IMDb API

🗣️ 自然语言处理

通过 Google Gemini AI 将用户的中英文问题转换为精确的 SQL 查询语句

🔄 智能错误恢复

SQL 执行失败时自动重试，智能修正查询语句，确保查询成功率

📡 流式响应

支持服务器发送事件(SSE)，实现实时的逐字符响应流

🎯 智能查询优化

自动处理重复标题、时间相关查询、多语言支持等复杂场景

🔗 外部 API 集成

集成 OMDb API 获取电影海报、详细信息等补充数据

💾 会话管理

维护完整的对话历史，支持上下文相关的连续查询

🚀 核心 API 端点： /api/chat (对话)、/api/chat/stream (流式)、/api/info (电影详情)、/api/history (历史)
